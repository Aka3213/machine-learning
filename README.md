# Titanic Survival Prediction 🚢

Predict survival on the Titanic using passenger data.

## 🧠 Problem Statement
Given features such as age, sex, class, etc., predict whether a passenger survived the Titanic disaster.

## 📁 Dataset
- [Kaggle Titanic Dataset](https://www.kaggle.com/c/titanic/data)
- Place `train.csv` and `test.csv` inside the `data/` folder.

## 🛠️ Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Joblib (for model saving)

## ⚙️ How to Run
1. Clone this repo
2. Install dependencies:  
   ```
   pip install -r requirements.txt
   ```
3. Run the model script:  
   ```
   python titanic_model.py
   ```

## ✅ Outputs
- Model accuracy
- Confusion matrix & classification report
- `titanic_model.pkl` saved for deployment
